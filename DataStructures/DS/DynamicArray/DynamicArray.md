# Dynamic Array

## Complexity

|                            | Time           |
| ---                        | ---            |
| Acess                      | Θ(1)           |
| Search                     | O(n)           |
| Insert/delete at beginning | Θ(n)           |
| Insert/delete at end       | Θ(1) amortized |
| Insert/delete in middle    | Θ(n)           |
